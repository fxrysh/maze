# Maze

A maze game written in pure WAT (WebAssembly Text).

## Controls

* `W` key - Move foward
* `A` key - Rotate counter clockwise
* `S` key - Move backward
* `D` key - Rotate clockwise
* Mouse - Rotate

> **NOTE**: For `A` and `D` key, please refer to [this](https://github.com/fxrysh/maze/issues/2) issue.
